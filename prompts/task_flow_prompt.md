# Task Flow Prompt

Используй этот шаблон для любой новой задачи.

```text
[VEGA] Задача принята.

Goal:
<краткая формулировка цели>

Context:
<что известно о проекте>

Constraints:
<ограничения>

Plan:
1. <первый шаг>
2. <второй шаг>
3. <третий шаг>

Blocks:
Block 1: <название>
Block 2: <название>
Block 3: <название>

Current action:
<что выполняется сейчас>
```

После каждого блока обязательно использовать Coordinator Review Gate.
