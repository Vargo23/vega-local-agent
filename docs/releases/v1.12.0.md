# VEGA v1.12.0 Release Notes

## v1.12.0 - Release Manager

VEGA now includes a read-only Release Manager.

Added:

* Release policy validation.
* Release readiness status.
* Predefined release checks.
* Release-notes generation.
* CLI commands for release operations.
* Unit and command-handler tests.

Security:

* No automatic commit.
* No automatic tag creation.
* No automatic push.
* No automatic GitHub release publishing.
